## Descripción del taller
Este proyecto corresponde al taller de **arreglos unidimensionales, bidimensionales y ArrayList** en Java.  
Se implementan diferentes ejercicios que permiten practicar el manejo de estructuras lineales, generación de datos aleatorios, procesamiento de información y aplicación de principios de programación orientada a objetos (Clean Code, DRY, YAGNI, KISS, SOLID).

El código está organizado en tres paquetes principales:
- `arreglos`: ejercicios con vectores (arreglos unidimensionales).
- `matrices`: ejercicios con matrices (arreglos bidimensionales).
- `arraylist`: ejercicios con listas dinámicas.

## ▶️ Instrucciones para ejecutar el programa
1. Clonar o descargar este repositorio desde GitHub:
   ```bash
   git clone https://github.com/usuario/Taller-Estructuras-Lineales.git
   - Abrir el proyecto en NetBeans o Visual Studio Code.
- Compilar el proyecto desde la carpeta src.
- Ejecutar los programas desde la consola:
- Cada ejercicio está numerado (Ejercicio1, Ejercicio2, etc.) dentro de su respectiva carpeta.
- Selecciona el archivo .java que quieras correr y ejecútalo con:
javac NombreDelArchivo.java
java NombreDelArchivo
